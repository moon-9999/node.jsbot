# 라이센스 파일이 없으면 봇이 켜지지 않습니다!

봇 기능은 도움말 EMBED를 참고해주세요!

config 폴더에 config.json 파일에 토큰과 접두사를 넣으시면 됩니다.
prefix는 접두사같은겁니다. Ex : "!" "/" 등..

그 외 index.js 수정은 마음대로 하시면 됩니다. (라이센스 코드 제외)
(index.js 파일에 주석을 확인하시면서 수정하세요!)
